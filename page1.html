What is the command line?

The command line is a user interface to used to navigate and interact with the computer

  • The command line accepts prompts or commands from the user
  • On mac and linux, the command line is the same as the terminal

What does it do?

It is very similar to a GUI (graphical user interface), like the file explorer, except instead of using your mouse to perform actions, you use commands

  • This allows for quicker access, more efficiency, and more elaborate commands to be performed

Commands to know

  • pwd
    • "pwd" stands for present working directory
    • This lets the user know where they currently are operating in the command line
  • ls/dir
    • "ls" and "dir" list all of the files within the current directory the user is in
    • When operating with prompt command prompt use "dir", and when operating with terminal use "ls"
  • cd
    • "cd" stands for change directory
    • This allows for the user to move between folders and directories
  • clear/cls
    • "clear" and "cls" are used to clear the previous entries into the command line/terminal from the textbox
    • This makes visualizing outputs much simplier, and can help clean up the screen if the text gets too messy
    • When using terminal, use "clear", and when using command prompt, use "cls"
  • mkdir
    • "mkdir" stands for make directory
    • This command allows us to easy create new folders/directories
    • To use this command, type "mkdir" followed by a space and the name you want for your directory
  • touch/echo
    • "touch" and "echo" are commands that allow us to make files from the command line
    • touch is for the terminal, and to use this command, type "touch" folloed by a space and the name you want for you file, but be sure to include the file type at the end (ex. "myFile.txt")
    • echo is a command that prints whatever you type after it, but by using the ">" operator with a file name, the user can "echo" a new file
      • ex. "echo > myFile.txt"
  • rm
    • "rm" stands for remove, and allows the user to remove files
    • to use the rm command, type "rm" followed by a space and the file you would like removed
      • ex. "rm myFile.txt"
  • mv
    • "mv" stands for move, and allows us to move a file to another location
    • To use the mv command, type "mv" followed by a space and the file you want moved and the path to the new location
      • ex. "mv myFile.txt" /Users/name/Desktop

Try it for yourself. Find what directory you are in, what files there are, and how to get to the next one.

Hint: "pwd" will tell you where you are, "dir" will let you know what's in your directory, and cd will take you to the next directory.
